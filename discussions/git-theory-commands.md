#### Command List

`$ git —version`
`$ git config user.name`
`$ git config —global user.email`
`$ git init`
`$ touch`
`$ git add`
`$ git status`
`$ git commit -m`
`$ git diff`
`$ git diff —staged`
`$ git reset`
`$ git remote add origin`
`$ git push origin master`
`$ git push -u origin master`
`$ git fetch`
`$ git show master`
`$ git log`
`$ git branch`
`$ git branch planet master^`
`$ tree .git/refs`
`$ git pull origin master`
`$ git push`
`$ git fetch`
`$ git pull`
`$ git checkout -b`
`$ git push origin planet —set-upstream`
`$ git clone`
`$ git branch -d planet`
`$ git branch -vv`
`$ git branch -no—merged`
